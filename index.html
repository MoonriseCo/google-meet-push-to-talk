Welcome to Google Meet Push To Talk 2.0

I built this Chrome Extension because I knew there was a need for an easier way to mute and unmute on Google Meet calls. Now we have over a million users around the world and we’ve added a bunch of new features. With this big 2.0 update I wanted to highlight what’s new:

  1. Language Support - It should now work regardless of language
  2. Mute from any Chrome tab using <Ctrl>-<Space>
  3. Other bug fixes and design refresh
  4. We've also integrated with the incredibly cool mütesync. It's a physical, light-up mute button that makes muting even easier.
